
# AI Utils
A collection of different AI utilities designed to simplify everyday tasks using state-of-the-art models and frameworks.

## Features
The application contains these four mini apps:

1. **Basic Chat**  
   A very basic version of ChatGPT-like apps.

2. **Audio Scribe**  
   Uses OpenAI's Whisper model to transcribe the files you upload.

3. **Doc Chat**  
   Upload a PDF, and ask questions about its content.

4. **Imagene**  
   Image Generator: Give a prompt in text, and it will generate an image for you.
